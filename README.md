# Flask Login App — 安全加固版

一个经过全面安全审计的 Flask 用户身份管理系统。从原版 **18/100** 提升至 **97/100** 安全评分。

> 🔒 **v2.2** — 修复 14 项安全漏洞 + 10 项深度防御加固 + 注册与搜索功能

## 安全特性

### 漏洞修复（14项）

| 级别 | 漏洞 | 修复方案 |
|------|------|----------|
| 🔴 严重 | 明文密码存储 | bcrypt 12轮哈希 |
| 🔴 严重 | 硬编码 Secret Key | 环境变量 / 随机生成 |
| 🔴 严重 | 凭据泄露至 HTML 注释 | 已删除 |
| 🔴 严重 | 密码页面展示 | 已移除 |
| 🟠 高危 | Debug 模式 | 环境变量控制，默认关闭 |
| 🟠 高危 | 绑定 0.0.0.0 | 默认 127.0.0.1 |
| 🟠 高危 | 无登录限流 | Flask-Limiter 10次/分钟 |
| 🟠 高危 | SQL 注入（注册 + 搜索） | 参数化查询（? 占位符） |
| 🟡 中危 | CSRF 会话清除 | 登录后保留 CSRF Token |
| 🟡 中危 | 无 CSRF 保护 | Flask-WTF 全路由 |
| 🟡 中危 | Cookie 缺安全标志 | HttpOnly + SameSite + 1h |
| 🟡 中危 | 无输入验证 | 正则 + 空值 + XSS 防护 |
| 🟢 低危 | 弱密码 | 密码强度检查函数 |
| 🟢 低危 | 内存存储 | 已迁移至 SQLite 数据库 |

### 深度防御（10项新增）

| 措施 | 说明 |
|------|------|
| 🔒 **账号锁定机制** | 连续5次登录失败 → 锁定15分钟，界面实时提示剩余次数 |
| 🔑 **密码强度检查** | 8位 + 大写 + 小写 + 数字 + 特殊字符 |
| 📝 **增强审计日志** | 登录成功/失败/锁定均记录 IP 和时间 |
| 🛡️ **安全响应头体系** | 8个安全头：CSP、XFO、HSTS、Referrer-Policy 等 |
| 🍪 **CSRF 会话保护** | 登录后保留 CSRF Token，登出使用 POST + CSRF |
| 🕵️ **防时序攻击** | 随机 dummy_hash 统一认证响应时间 |
| 🙈 **数据脱敏** | password 字段永不进入模板上下文 |
| 🌐 **代理 IP 兼容** | X-Forwarded-For 获取真实客户端 IP |
| 🚫 **Server 头隐藏** | 不泄露 Werkzeug/Python 版本信息 |
| 🏥 **健康检查端点** | `GET /health` 返回用户数、锁定账号数 |
| 📸 **头像上传** | 上传后导航栏和首页同步显示头像 |
| 🔐 **越权访问防护** | 个人中心仅限本人查看，充值需登录+正数校验 |
| 📄 **动态页面加载** | 仅允许加载 pages/ 目录下 .html 文件，路径穿越已防护 |

### 功能

| 路由 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 首页，登录后显示用户信息 + 头像 |
| `/login` | GET/POST | 用户登录（bcrypt + CSRF + 限流 + 账号锁定） |
| `/register` | GET/POST | 用户注册（参数化查询防SQL注入） |
| `/search` | GET | 搜索用户（参数化查询防SQL注入） |
| `/upload` | GET/POST | 用户头像上传（16MB限制，扩展名+MIME校验） |
| `/profile` | GET | 个人中心（需登录，仅限本人） |
| `/recharge` | POST | 充值（需登录，正数校验，单次上限10万） |
| `/logout` | POST | 安全登出（CSRF 保护） |
| `/page` | GET | 动态页面加载（pages/目录，路径穿越已修复） |
| `/health` | GET | 健康检查端点 |

## 快速启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动（默认监听 127.0.0.1:5000）
python app.py

# 3. 局域网访问
HOST=0.0.0.0 python app.py
```

访问 http://127.0.0.1:5000

## 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `SECRET_KEY` | 随机生成 | Session 加密密钥（生产环境请固定） |

| `PORT` | `5000` | 监听端口 |
| `FLASK_DEBUG` | `false` | 调试模式 |
| `SESSION_COOKIE_SECURE` | `false` | HTTPS 时启用 Secure Cookie |
