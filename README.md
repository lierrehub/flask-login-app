# Flask Login App

一个安全的 Flask 用户登录演示应用，包含用户认证和会话管理功能。

> ⚠️ **安全版本**：本应用已修复 13 项安全隐患，适合作为安全 Flask 应用的参考模板。

## 安全特性

| 防护措施 | 说明 |
|----------|------|
| 🔒 **密码哈希** | 使用 bcrypt 加密存储，源码零明文密码 |
| 🔑 **密钥管理** | 通过环境变量配置，支持随机回退 |
| 🛡️ **CSRF 保护** | Flask-WTF 全路由自动校验（含登出接口） |
| 🚫 **登录限流** | 每分钟最多 10 次尝试，防止暴力破解 |
| 🍪 **Session 安全** | HttpOnly + SameSite + 1小时过期 |
| 🔐 **会话旋转** | 登录后刷新 session，防会话固定攻击 |
| 📋 **安全响应头** | CSP + X-Frame-Options + X-Content-Type-Options |
| ✅ **输入验证** | 用户名格式校验 + 空值检查 |
| 🕵️ **防枚举** | 时序攻击防护，无法探测用户是否存在 |

## 快速启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 按需配置环境变量
cp .env.example .env
# 编辑 .env 文件修改配置

# 3. 启动（默认监听 127.0.0.1:5000）
python app.py
```

访问 http://127.0.0.1:5000

## 环境变量（可选）

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `SECRET_KEY` | 随机生成 | Session 加密密钥（生产环境请固定） |
| `HOST` | `127.0.0.1` | 监听地址（生产可改为 `0.0.0.0`） |
| `PORT` | `5000` | 监听端口 |
| `FLASK_DEBUG` | `false` | 调试模式（生产环境不要开启） |
| `SESSION_COOKIE_SECURE` | `false` | 启用 Secure cookie（需要 HTTPS） |
